# 中国人民大学自动完成课程作业程序

## 使用方法
- 直接右上角点击download
- 双击打开 /dist/autotask.exe;
- 登录账号密码.
- 点击开始自动作业

>> NOTIC:
	请求速度有所控制，防止ip被封。
	运行错误，请尝试重试，重试还是不行请等待5-10分钟后在再次尝试。
